# stealth_audio_recorder.shortcut
IOS Shortcut for covertly recording audio on your IPhone

A few weeks ago, my shit-for-brains psycho neighbor first tried to run me over with his car and then verbally abused me in the harshest possible way. During this incident, I was wearing Bluetooth headphones and had no easy way to turn them off and start recording this “charming” argument to use in court.

This iOS shortcut solves the problem for any future incidents. Tapping the shortcut stops all audio playback and disables Bluetooth. After the phone’s screen dims, recording begins. Tapping the screen again stops the recording, re-enables Bluetooth, and sets the phone’s brightness to 50%. The finished recording can then be found in the “Shortcuts” folder in iCloud.
